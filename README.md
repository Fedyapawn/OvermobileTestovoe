# Unity Mobile Sticky Leaderboard

Этот проект реализует  **Leaderboard** для мобильных игр на Unity. Главная особенность — «липкий» элемент игрока, который прижимается к верхнему или нижнему краю списка, когда основной объект уходит из зоны видимости.

##  Техническая реализация

### Как работает "Липкость"?
Скрипт использует метод `GetWorldCorners()` для получения точных координат углов вьюпорта и элемента в мировом пространстве. 
1. Как только **нижняя** граница игрока пересекает **нижнюю** границу вьюпорта — включается нижний Sticky-дубликат.
2. Как только **верхняя** граница игрока пересекает **верхнюю** границу вьюпорта — включается верхний Sticky-дубликат.
3. В момент активации дубликата, оригинальный объект в списке становится прозрачным (`alpha = 0`), сохраняя свое место в макете.
